# LibreStock Inventory Web

This is the TanStack Start frontend for LibreStock Inventory, built with Vite, TanStack Router, and React.

## Getting Started

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Build and Run

```bash
pnpm build
pnpm start
```

## Project Notes

- File-based routes live in `src/app`.
- The router is configured in `src/router.tsx` and generated in `src/routeTree.gen.ts`.
- Environment variables use `VITE_` prefixes (see `env.template`).
