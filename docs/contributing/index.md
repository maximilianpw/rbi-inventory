# Contributing

Thank you for your interest in contributing to LibreStock Inventory! This section covers how to contribute to the project.

## Ways to Contribute

- **Report bugs** - Found an issue? [Open an issue](https://github.com/your-org/rbi/issues)
- **Suggest features** - Have an idea? Share it with us
- **Submit code** - Fix bugs or add features
- **Improve documentation** - Help others understand the project

## Getting Started

1. **Read the guidelines** - Review our [contribution guidelines](guidelines.md)
2. **Set up your environment** - Follow the [development setup](../development/setup.md)
3. **Find an issue** - Look for `good first issue` labels
4. **Submit a PR** - Follow our [pull request process](pull-requests.md)

## Code of Conduct

We are committed to providing a welcoming and inclusive environment. Please be respectful and constructive in all interactions.

## Questions?

If you have questions about contributing:

- Check existing [issues](https://github.com/your-org/rbi/issues)
- Review the [documentation](../index.md)
- Ask in discussions
