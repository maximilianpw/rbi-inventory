# Démarrage

Cette section vous aidera à installer et configurer LibreStock Inventory.

## Prérequis

Avant de commencer, assurez-vous d'avoir :

- Node.js >= 20.0.0
- pnpm >= 10.0.0
- Nix avec flakes activé (pour devenv)
- PostgreSQL 16 (fourni par devenv)
